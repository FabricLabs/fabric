Playnet / federation — chain flush

Sends Fabric wire message P2P_FLUSH_CHAIN (opcode 0x56) with a JSON body { snapshotBlockHash, network, label? } to every connected Fabric peer whose registry score is strictly greater than 800 (see Peer.settings.flushChainMinTrustedScore). Each recipient validates the sender the same way, rewinds its bitcoind to snapshotBlockHash via invalidateblock, and relays again only to highly trusted peers.

Wire this button to your Hub process: call fabricPeer.sendFlushChainToTrustedPeers({ snapshotBlockHash, network, label }) on your Peer from @fabric/core. This file is static HTML only.
